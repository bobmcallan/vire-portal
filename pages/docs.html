WHAT IS VIRE

Vire is an investment portfolio tracking platform. It connects to your brokerage data via Navexa and exposes portfolio analytics through a web dashboard and an MCP (Model Context Protocol) endpoint for Claude.

With Vire you can view holdings, track capital performance, monitor technical indicators, and query your portfolio data conversationally through Claude Desktop or Claude Code.

NAVEXA

Vire syncs portfolio data from Navexa, an investment portfolio tracker that aggregates holdings, transactions, and performance metrics from brokers and exchanges.

You need a Navexa account and API key to use Vire. If you do not have a Navexa account, create one at navexa.com.

GETTING YOUR API KEY

  1. Sign in to your Navexa account at navexa.com
  2. Open Account Settings
  3. Navigate to the API section
  4. Generate or copy your API key

CONFIGURING VIRE

Once you have your Navexa API key:

  1. Go to Profile in Vire
  2. Paste your Navexa API key into the API KEY field
  3. Click SAVE

Vire will sync your portfolios automatically. Return to the Dashboard to view your holdings and performance data.
